# jobze-site

Landing page and waitlist capture for JOBZE, a Slack-native AI job search agent. Live at [jobze.io](https://jobze.io).

Also hosts the WVEZ portfolio infographics (RECON, FLIPPR, CRP), served from this repo and linked from [wvez.org](https://wvez.org).

Vanilla HTML/CSS/JS. IBM Plex Mono, Inter, and Syne type stack. Hosted on GitHub Pages.

## Contents

**JOBZE**
- `index.html` — product landing page
- `mobile.html` — mobile-optimized variant
- `profile.html` — Phase 2 deep profile form for waitlist signups

**WVEZ portfolio infographics**
- `WVEZ REVAMP RECON.html`
- `WVEZ REVAMP FLIPPR.html`
- `WVEZ CRP.html`

**Assets**
- `logo.glb` — 3D pendant rendered via model-viewer
- Brand images, favicons, site manifest
